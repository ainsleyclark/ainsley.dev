Components

This selection of components makes up the ainsley.dev brand and are the building blocks of any material or site asset.

Buttons

Buttons should appear with an arrow next to them. If placed on a dark background, a white button should be used and visa versa.

Small tags are used for categorising content for blog posts and articles. As with buttons, they should be placed with the background colour in mind.

Arrow

Using the ainsley.dev arrow can be a convenient way to navigate the user to additional content that’s not on the current page. Ensure the arrow is proportionate in size to the heading and is not too large or small.
